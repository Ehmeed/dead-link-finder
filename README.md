# Dead link finder

Find links that don't work

## Build
Compile executables:
```bash
./gradlew build
```
Native located in `./build/bin/native/debugExecutable/dead-link-finder.kexe`

JVM located in `./build/libs/dead-link-finder-jvm-{version}-SNAPSHOT.jar`
## Testing
To run all tests:
```bash
./gradlew allTests
```
